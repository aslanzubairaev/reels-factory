# Photo Library

Библиотека фото для обложек Reels.

## Категории

### work/ — Рабочие фото
Для обложек про: программирование, AI, инструменты, code, сетап.
- За ноутбуком, в наушниках, с кодом на экране.

### portrait/ — Портреты
Для обложек про: мотивацию, личные истории, перезапуск.
- Крупный план, селфи, нейтральный фон.

### lifestyle/ — Лайфстайл
Для обложек про: дисциплину, продуктивность, путешествия, вайб.
- Машина, дорога, города, природа.

## Использование
Пайплайн автоматически выбирает фото из нужной категории
на основе `content_category` из анализа контента.
