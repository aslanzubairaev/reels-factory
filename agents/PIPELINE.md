# Reels Factory — Pipeline

> Полный процесс создания Reel от идеи до видео.

## Как это работает

Ты общаешься с Claude Code в терминале Cursor. Он управляет агентами, которые помогают тебе на каждом шаге. Ни один шаг не выполняется без твоего утверждения.

## Шаги

### 1. Настройка профиля (опционально)

Скажи: **«Настрой профиль»**

Claude спросит про твою нишу, аудиторию, тон и стиль. Создаст `config/profile.md`. Другие агенты будут учитывать этот профиль.

Можно пропустить — тогда каждый раз указываешь параметры вручную.

### 2. Поиск идей

Скажи: **«Найди идеи для Reels про [тему]»**

Claude найдёт тренды через веб-поиск и предложит 5-10 идей. Каждая идея содержит суть, пользу для аудитории, психологию и оценку потенциала.

Ты выбираешь идею, редактируешь или предлагаешь свою.

### 3. Сценарий

Скажи: **«Напиши сценарий на русском»**

Claude предложит 2-3 варианта сценария с разными подходами. Ты выбираешь лучший. Сценарий содержит текст для телепромптера, layout каждой части и промпты для генерации фонов.

Также получишь: анализ хука, варианты CTA и хэштеги.

### 4. Генерация фонов

Скажи: **«Сгенерируй фоны»**

Claude сгенерирует 2-3 варианта фона для каждой части через AI. Ты выбираешь лучший.

### 5. Запись в студии

Запусти студию:
```bash
node studio/server.js
# Открой http://localhost:3000
```

В студии: выбери проект → настрой камеру → посмотри превью всех слайдов → запиши видео.

### 6. Результат

Готовое видео в `projects/[name]/output/` — MP4, 1080×1920, 30 FPS. Загрузи в Instagram.

## Горячие клавиши студии

| Клавиша | Действие |
|---------|----------|
| ← → | Навигация по слайдам |
| Space | Запись / Стоп |
| R | Перезапись части |
| Esc | Назад к Preview |
