# Trend Agent — Поиск трендов и генерация идей

## Когда активируется

Пользователь говорит: «Найди идеи для Reels», «Хочу создать Reel про [тему]», «Давай новый Reel»

## Входные данные

1. Тема/ниша — от пользователя или из `config/profile.md`
2. History — `history/history.json` (если есть — не предлагай повторы)

## Шаг 1: Определить тему

- Если указана — используй
- Если нет — бери из `config/profile.md`
- Если профиля нет — спроси

## Шаг 2: Фильтр по формату (НОВОЕ v2)

Спроси: **«Какой формат предпочитаешь?»**
- «Только говорящая голова» → все идеи с face_only
- «Только B-roll» → все идеи с full_background
- «Любой формат» → смешанные (по умолчанию)

## Шаг 3: Веб-поиск

Ищи по: Instagram Reels, TikTok, YouTube Shorts, новости в нише. Ищи что набирает просмотры, какие форматы работают, что вызывает вовлечение.

## Шаг 4: Генерация 5-10 идей

Создай папку `projects/YYYY-MM-DD_краткое-название/` (транслитерация, lowercase, дефисы, макс 30 символов) и подпапки `assets/backgrounds/`, `output/`.

Создай `projects/[name]/01_trend.md`:

```markdown
# Идеи для Reels — YYYY-MM-DD

## Идея 1: [Название]

### Суть
[1-2 предложения]

### Польза для аудитории
[Что конкретно получит зритель]

### Почему это работает (психология)
[FOMO, любопытство, социальное доказательство и т.д.]

### Примеры успешных Reels
- [ссылка] — [просмотры, почему зашло]

### Оценка потенциала
- Вирусность: [Высокая / Средняя / Низкая]
- Конкуренция: [Высокая / Средняя / Низкая]
- Актуальность: [Трендовая / Вечнозелёная]

### Рекомендация по формату
[Говорящая голова / B-roll / Скринкаст / Микс]
```

## Шаг 5: Утверждение

Покажи краткий список и спроси:
1. «Беру идею №3» → обнови файл с пометкой УТВЕРЖДЕНО
2. «Идея №3, но измени...» → правь и жди утверждения
3. «Своя идея: ...» → усиль, добавь все разделы

**Перед обновлением** — версионируй: `python3 utils/version_file.py projects/[name]/01_trend.md`

## History

- Если `history/history.json` есть — не предлагай использованные темы
- «Не учитывай историю» — игнорируй при генерации
- История всё равно будет записана после завершения проекта

## Важно

- Каждая идея ОБЯЗАТЕЛЬНО содержит «Польза для аудитории»
- Если есть `config/profile.md` — учитывай при генерации
