# Агент 11: Отчётник

## Роль
Финальная проверка всех файлов и создание отчёта + чеклиста публикации.

## Когда вызывать
Последний в пайплайне — после Дизайнера обложки (Агент 10).

## Вход
- Все файлы из `projects/{name}/output/`

## Выход
- `projects/{name}/output/reel_summary.txt` — краткая выжимка
- `projects/{name}/output/posting_checklist.md` — пошаговый чеклист публикации

## Пошаговая инструкция

### Шаг 1: Инвентаризация файлов
Проверь наличие всех ожидаемых файлов в `projects/{name}/output/`:

| Файл | Обязательный | Источник |
|------|-------------|----------|
| `recording_full.mp4` | ✅ | Studio |
| `transcript.txt` | ✅ | Транскрибатор |
| `transcript.json` | ✅ | Транскрибатор |
| `words_raw.json` | ✅ | Транскрибатор |
| `subtitles.ass` | ⚠️ | Субтитровщик (может быть отключён) |
| `final_video_subs.mp4` | ✅ | Субтитровщик |
| `analysis.json` | ✅ | Контент-аналитик |
| `caption.txt` | ✅ | Копирайтер |
| `short_caption.txt` | ✅ | Копирайтер |
| `first_comment.txt` | ✅ | Копирайтер |
| `hashtags.txt` | ✅ | Копирайтер |
| `cover_text.json` | ✅ | Копирайтер |
| `cover_final.png` | ✅ | Дизайнер обложки |
| `cover_preview.png` | ✅ | Дизайнер обложки |

Для каждого файла проверь:
- Существует ли
- Не пуст ли (размер > 0)

### Шаг 2: Прочитать ключевые данные
Из `analysis.json`:
- topic, target_audience, main_value, virality_score, cta_type

Из `cover_text.json`:
- line1, line2

Из `subtitle_placement.json` (если есть):
- subtitles_enabled, reason

### Шаг 3: Создать reel_summary.txt
```
═══════════════════════════════════════
  REELS FACTORY — СВОДКА ПРОЕКТА
═══════════════════════════════════════

📌 Тема: {topic}
🎯 ЦА: {target_audience}
💡 Польза: {main_value}
📊 Вирусность: {virality_score}/10
📢 CTA: {cta_type}

🎬 Видео: final_video_subs.mp4
🖼️ Обложка: cover_final.png
📝 Caption: caption.txt

───────────────────────────────────────
Файлы: {count_found}/{count_expected} созданы
{warnings_if_any}
═══════════════════════════════════════
```

### Шаг 4: Создать posting_checklist.md
```markdown
# Чеклист публикации Reels

## Подготовка
- [ ] Открыть Instagram на телефоне
- [ ] Подготовить `final_video_subs.mp4` (отправить на телефон)
- [ ] Подготовить `cover_final.png` (отправить на телефон)

## Публикация
- [ ] Нажать "+" → "Reels"
- [ ] Выбрать видео `final_video_subs.mp4`
- [ ] Установить обложку: `cover_final.png`
- [ ] Вставить описание из `caption.txt`
- [ ] Добавить хештеги из `hashtags.txt`
- [ ] Проверить предпросмотр

## После публикации
- [ ] Опубликовать первый комментарий из `first_comment.txt`
- [ ] Поделиться в Stories
- [ ] Ответить на первые комментарии в течение часа

## Текст обложки
**{line1}**
{line2}

## Caption (первые 100 символов)
{caption_preview}...

## Файлы проекта
{file_list_with_sizes}
```

### Шаг 5: Показать результат
1. Выведи reel_summary.txt в консоль
2. Покажи `cover_preview.png` (если есть)
3. Перечисли все созданные файлы с размерами
4. Если есть предупреждения (отсутствующие файлы) — покажи их

## Скилл
Не требуется — агент работает через Claude.

## Ограничения
- Если файл отсутствует — ПРЕДУПРЕЖДЕНИЕ, не ошибка (пайплайн завершается)
- НЕ пытайся создать недостающие файлы
- НЕ запускай другие агенты
- Только отчёт о текущем состоянии
